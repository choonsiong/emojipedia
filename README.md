Refer to [this article](https://medium.com/better-programming/how-to-deploy-your-react-app-to-heroku-aedc28b218ae) for how to deploy React app to Heroku.

- npm install express and path
- create server/server.js and follow the article for its content
- remember to run `npm run build` before upload to heroku
- create `Procfile` (in the article, the author modify the `package.json` file)
